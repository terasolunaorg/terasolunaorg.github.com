What is TERASOLUNA Server Framework for Java (5.x)?

This framework is a comprehensive guide for WHAT, WHERE, WHEN, WHY and HOW for web development with (SpringMVC + Spring + MyBatis3/JPA) based technology stack. The primary purpose of this framework is to make it easier to develop spring based applications by providing the know-how which can only be achieved with years of experience in scalable java based web development.

Artifacts

TERASOLUNA Server Framework for Java (5.x) consists of following artifacts:

Development Guideline

Step-by-Step Tutorials and intensive and focused best practices which take in account numerous real world particular scenarios not to be found anywhere on the internet or reference books. See Development Guideline.

Common Library

This is a library of useful and obstrusive common functionalities. Refer to terasoluna-gfw to for further details about common library.

Blank Project

Blank template to quickly start with a solid foundation of project configuration and packing structure. Refer to the following projects to get started with Blank Project.

  • terasoluna-gfw-web-multi-blank (Recommended)
  • terasoluna-gfw-web-blank

Sample Application

Working example of all the best practices advised in development guideline. Take inspiration directly from the code. Refer to the following projects for further details to get started with Reference Application.

  • terasoluna-tourreservation (MyBatis3)
  • terasoluna-tourreservation (JPA)
